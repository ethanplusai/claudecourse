# Pre-Launch Checklist

Run through every item before you go live. Don't skip anything.

## Platform
- [ ] Landing page is live and loads correctly
- [ ] Landing page copy matches your offer and audience
- [ ] Signup flow works (test with email AND phone)
- [ ] Login flow works
- [ ] Portal shows modules and progress correctly
- [ ] All lessons have content (no placeholder text)
- [ ] Sequential gating works (can't skip ahead)
- [ ] "Mark as Complete" button works
- [ ] Progress bar updates after completing lessons
- [ ] Mobile responsive — test on your phone
- [ ] Custom domain is connected
- [ ] SSL certificate is active (https works)

## Nurture System
- [ ] CRM is set up and receiving contacts
- [ ] New signups appear in CRM automatically
- [ ] Welcome email/SMS sends within 60 seconds of signup
- [ ] Email sequence is loaded and triggers are set
- [ ] SMS messages are configured (if using)
- [ ] Behavior tags are firing (lesson completions, CTA clicks)
- [ ] Re-engagement automation is active
- [ ] Unsubscribe/opt-out works correctly

## Booking
- [ ] Calendar is set up with correct availability
- [ ] Timezone handling works
- [ ] Booking confirmation email sends
- [ ] Reminder sequence is active (24hr + 1hr)
- [ ] Pre-call questionnaire is attached
- [ ] Booking CTA appears in the right places

## Traffic
- [ ] At least one traffic source is ready to go
- [ ] Outreach messages are written
- [ ] Landing page URL is in all outreach/content
- [ ] Tracking is set up (know where signups come from)

## End-to-End Test
- [ ] Sign up as a new user
- [ ] Receive welcome email/SMS
- [ ] Complete 2-3 lessons
- [ ] Verify behavior events are tracked
- [ ] Receive lesson completion follow-up
- [ ] Go quiet for 2 days — receive re-engagement message
- [ ] Click booking CTA — booking page works
- [ ] Book a test call — receive confirmation + reminders

## Go Live
- [ ] All tests pass
- [ ] First batch of outreach/content is queued
- [ ] Monitoring is set up (check daily for first week)
- [ ] Launch.
